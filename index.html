Ravenous

Ravenous is a React-based web app that uses the Yelp API to help users search for local businesses. It has a modern and intuitive design that allows users to search by location, category, price range, and rating. The app presents search results in a clear and informative way, including business name, address, phone number, website, and user reviews. It also allows users to save businesses to a list and view detailed information such as photos and reviews. The tech stack includes React, React-Router, and Yelp API, and the app will be deployed on a web server for easy access.
